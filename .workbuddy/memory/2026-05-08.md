## 完成：跨平台 Socket 抽象层

### 新增文件
- `include/asyncio/socket/socket_os.h` - 平台特定头文件选择（Windows/macOS/Linux）
- `include/asyncio/socket/socket_base.h` - Socket 基类、TcpSocket、TcpListener、UdpSocket、SockAddr
- `include/asyncio/socket/socket_ops.h` - 异步 Socket 操作接口
- `src/socket/socket_base.cc` - Socket 基类实现
- `src/socket/socket_ops.cc` - 异步操作实现
- `tests/socket_test.cc` - Socket 测试

### 核心 API
```cpp
// TCP
std::shared_ptr<TcpSocket> TcpSocket::Create();
Future<void> TcpSocket::Connect(host, port);
Future<std::vector<uint8_t>> TcpSocket::Recv(max_len);
Future<size_t> TcpSocket::SendAll(data);  // 确保全部发送

// Listener
std::shared_ptr<TcpListener> TcpListener::Create();
Future<void> TcpListener::Bind(host, port);
void TcpListener::Listen(backlog);
Future<std::tuple<std::shared_ptr<TcpSocket>, SockAddr>> Accept();

// UDP
std::shared_ptr<UdpSocket> UdpSocket::Create();
Future<std::tuple<std::vector<uint8_t>, SockAddr>> RecvFrom(max_len);
Future<size_t> SendTo(data, addr);
```

### 跨平台支持
- Windows: Winsock2 (SOCKET, WSAStartup)
- macOS: BSD sockets
- Linux: POSIX sockets
